在安装MySQL时失败，可以尝试一下一下解决办法：

    $ sudo apt-get remove mysql*
    $ sudo apt-get autoremove mysql*
    $ rm -rf /etc/mysql
    $ rm -rf /var/lib/mysql
    $ mv /var/lib/dpkg/info /var/lib/dpkg/info.bak
    $ mkdir /var/lib/dpkg/info
    
再次尝试安装

    $ sudo apt-get install mysql-server mysql-client
